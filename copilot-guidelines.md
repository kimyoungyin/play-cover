# GitHub Copilot Guidelines

답은 항상 한국어로 해줘. 단계별로 차근차큰 알려줘

이 프로젝트에서 나는 악기 연주자들이 특정 곡에 연주한 커버 녹음 파일을 업로드하면 그 원곡 유튜브와 같이 재생되서 사용자가 여러 명의 커버곡을 들어보고 투표하는 웹앱을 만들고 싶어

## 개발 스택

-   Next.js(14)
-   Typescript
-   mysql(AWS RDS)
-   AWS S3
-   배포: Vercel
-   그 외 추가 가능

## 필요한 주요 기능

-   로그인/회원가입 기능: 회원가입을 하지 않아도 커버 음원 감상은 가능하지만 본인 커버곡 업로드나, 다른 음원 좋아요/싫어요는 불가능하다.
-   유튜브 음악 영상 검색 기능: 음악(노래) 카테고리에 있는 영상 검색
-   유튜브 영상마다 녹음 파일 업로드 기능: 검색해서 나온 영상에 커버 녹음을 업로드하겠다고 결정하여 그 영상 옆 커버 음원 업로드를 클릭하면, 연주자가 미리 녹음해둔 파일을 영상 당 하나 올릴 수 있는 form이 등장한다. 미리 듣기 후 최종 업로드 시 S3에 해당 녹음파일이 저장됨과 동시에 유저가 올린 커버곡 정보가 RDS에 업로드된다.
-   유튜브 영상에 대응되는 커버 녹음을 영상과 동시 재생하여 감상 가능: 유튜브 음악 영상 하나를 유저가 클릭해서 들어가면 유튜브 영상이 좌측에, 이에 해당하는 커버 음원 목록이 우측에 목록으로 뜬다. 음원 중 하나를 클릭하면 유튜브 영상과 음원이 동시에 재생되어 커버곡을 감상할 수 있다.
-   커버 음원 좋아요/싫어요 기능: 음원이 좋거나 싫은 경우 음원마다 좋아요/싫어요를 할 수 있다.
-   유저 프로필: 접근하면 유저가 올린 커버 음원 목록을 확인하고 들어볼 수 있다.

## 추후 추가 기능

-   유저마다 커버 음원 업로드 수 제한: 5회
-   구독 결제 기능: 구독하면 횟수 무제한 업로드 가능
